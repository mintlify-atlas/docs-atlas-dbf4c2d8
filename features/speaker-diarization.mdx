whisper.cpp supports two approaches to identifying different speakers in audio: tinydiarize for detecting speaker turns in a single audio stream, and stereo diarization for separating two speakers from a stereo file.

## tinydiarize

tinydiarize is a local speaker turn detection approach that runs entirely on-device. It does not identify who is speaking — it marks where one speaker stops and another begins, inserting `[SPEAKER_TURN]` tokens into the transcript.

<Warning>
  tinydiarize is an experimental feature. It requires a special model variant and may produce inaccurate results depending on the audio.
</Warning>

### Requirements

tinydiarize requires the `small.en-tdrz` model, which is a fine-tuned version of the `small.en` Whisper model trained to output speaker turn tokens.

### Setup

<Steps>
  <Step title="Download the tinydiarize model">
    ```bash
    ./models/download-ggml-model.sh small.en-tdrz
    ```
  </Step>
  <Step title="Run transcription with speaker turn detection">
    ```bash
    ./build/bin/whisper-cli -f ./samples/a13.wav -m ./models/ggml-small.en-tdrz.bin -tdrz
    ```
  </Step>
</Steps>

## Stereo diarization

For stereo audio files where each channel contains a different speaker, use the `-di` flag. whisper.cpp processes the left and right channels separately and interleaves the results:

```bash
./build/bin/whisper-cli -f stereo_audio.wav -m ./models/ggml-base.en.bin -di
```

This is useful for phone call recordings or interview audio where speakers are separated into channels.

## Limitations

<CardGroup cols={2}>
  <Card title="English only" icon="language">
    The `small.en-tdrz` model is English-only. There is no multilingual tinydiarize model.
  </Card>
  <Card title="Turn detection, not identification" icon="users">
    tinydiarize marks where speakers change but does not label or identify individual speakers.
  </Card>
  <Card title="Experimental accuracy" icon="flask-conical">
    Speaker turn boundaries may be imprecise, especially in overlapping speech or noisy audio.
  </Card>
  <Card title="Model size" icon="hard-drive">
    The `small.en-tdrz` model is larger than `tiny.en` or `base.en`. See the [models overview](/models/overview) for memory requirements.
  </Card>
</CardGroup>

For more background on the tinydiarize implementation, see [PR #1058](https://github.com/ggml-org/whisper.cpp/pull/1058).
