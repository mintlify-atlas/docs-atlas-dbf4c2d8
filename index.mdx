whisper.cpp is a port of [OpenAI's Whisper](https://github.com/openai/whisper) ASR model written in plain C/C++. It runs fully offline with zero external dependencies, supports Apple Silicon, NVIDIA GPUs, Vulkan, and more — and ships a clean C API you can embed in any application.

## Get started in minutes

<Steps>
  <Step title="Clone and build">
    ```bash
    git clone https://github.com/ggml-org/whisper.cpp.git
    cd whisper.cpp
    cmake -B build
    cmake --build build -j --config Release
    ```
  </Step>
  <Step title="Download a model">
    ```bash
    sh ./models/download-ggml-model.sh base.en
    ```
  </Step>
  <Step title="Transcribe audio">
    ```bash
    ./build/bin/whisper-cli -m models/ggml-base.en.bin -f samples/jfk.wav
    ```
  </Step>
</Steps>
